Settings frame

The "Settings" frame combines all jpiv settings in one place. Click the tree nodes in the left part of this window to expand them. Click on the different items in the tree to access different settings panels. All settings are saved in a file in the users home directory as soon as the input components like text fields or check boxes loose their focus (meaning: as soon as you click somewhere else).
