Settings - Preferences - Load/Save



settings-panel: Preferences - Load/Save.

write Tecplot header
If selected, all vector output files will have a tecplot style header. Example:
TITLE = "vectorfilename.jvc"
VARIABLES = "x", "y", "Vx", "Vy", "s"
ZONE I=75, J=75, F=POINT
