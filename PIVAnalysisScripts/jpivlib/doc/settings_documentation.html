Settings - Documentation



settings-panel: Documentation.

This panel holds a text area that is able to display files in html format and that can handle hyperlink events. When called, the program generates the following path: "file: + file separator + file separator + jpiv library path + file separator + doc + file separator + index.html" The file seperator is system dependend either "/" or "\". The jpiv library path is specified here.
