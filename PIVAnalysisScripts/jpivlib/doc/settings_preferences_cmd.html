Settings - Preferences - Cmd



settings-panel: Preferences - Cmd.

Maximum number of lines
Limits the size of the Cmd-history to the specified number of lines.
Enable line wrap
Eneables line wrapping. If not selected, a horizontal scroll bar will appear to show lines that are longer than the frame width.
